# Finance Reading Hub

A curated collection of books, essays and resources about investing,
markets and financial thinking.

The goal of this project is to organize high-quality material that helps
build a long-term understanding of finance, decision making and wealth
management.

This repository powers a simple static website published with GitHub Pages.

## Website

https://mberrueta.github.io/finance-reading-hub/

## Topics covered

- investing fundamentals
- market psychology
- long-term portfolio thinking
- financial decision making
- notable books and essays

## Why this project

Over time I collected many resources about investing and financial thinking.
This project organizes them into a simple and accessible reading hub.

## Tech

Simple static site hosted on **GitHub Pages**.
